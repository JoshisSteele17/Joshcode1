Command line OR Terminal SHORTCUTS

Up and Down Keys : Shows you your last commands

PWD : shows you where you are

LS : shows you a list of files in the current directory

TOUCH : create a file.

EXAMPLE: touch index.html

CD : Change Director

EXAMPLE: cd Desktop
NOTE: cd takes you to the top level directory

MKDIR : Make directory or folder

RM : removes a file

RM -RF : removes a directory or folder

GIT SHORTCUTS

GIT CLONE : clones your repo

EXAMPLE: git clone https://example.com

GIT STATUS : checks the status of our changes

EXAMPLE: git status

GIT ADD . : adds everything you changed

EXAMPLE: git add .

GIT ADD : adds a specific file

EXAMPLE: git add index.html

GIT COMMIT : commits the changes you want to push

EXAMPLE: git commit -m "changed index.html"

GIT PUSH : pushes the changes

EXAMPLE: git push
